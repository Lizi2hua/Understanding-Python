# 语义分割综述（Review of Semantic Segmentaion ）

语言所含的意义就是语义（semantic)。语义可以简单地看作是数据所对应的现实世界中的事物所代表的概念的含义，以及这些含义之间的关系，是数据在某个领域上的解释和逻辑表示。

## 1.[语义分割的基本介绍](https://zhuanlan.zhihu.com/p/37618829)

​	语义分割就是按照图像**像素**的语义来分类，可以简单理解为像素级别的分类任务。输出图片维度一般为**(*H×*W×class)**，每一个channel对应一个class,即每一个pixel位置都有class数目个channel，（**每个pixel的channel的值对应class的预测概率**？）。
